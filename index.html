  • Projects

Projects

At the moment my project porfolio is still in need of great improvement. I will describe my two favorite projects briefly.

The first was a database related project. I worked in a team of three people, and we made an interactive covid database. We made all of this in a shared AWS instance, and utilized several different tools. Python was our language of choice to implement everything. All the information was pulled from a github repository where the information was stored in an excel file. We would have a chronjob method that would pull and strip the information from excel daily. We normalized the database into several different tables making it 3nf. Mysql was used for the databse management. We then used flask to have our information on the web. We did so by utilizing different routes that connected to json functions we implemented. These json functions would gather specific information from the database based on what someone is looking for. For instance the amount of covid deaths in the zipcode 06443. We then implemented AJAX to have a nice flow of information.

The second project worth recognition was an app development in java. The project was conducted in a group of 4 people and we utilized a Scrum process. We worked in 2 week sprints developing features for a digitalized version of Machi Koro. During the project we all used github to have a repository to work from. We used junit for the unit testing, cucumber for the acceptance testing, jacoco for the code coverage and gradle for a build manager. This project really opened my eyes to my love for software engineering, and going through the problems of knowing how to utilize the correct amount of classes to get maximum efficiency. This was also a good experience because we were all remote due to covid. We also had a member of our group contributing from a country with a time difference of 7 hours. It made the meetings very interesting and the ability to pair program difficult at times. Overall, it was an amazing simulation of a real world work environment.
